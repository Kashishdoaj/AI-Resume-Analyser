# AI Resume Analyser

An AI-powered Resume Analyser built using Python and Streamlit that helps users evaluate their resumes, identify skill gaps, and improve ATS compatibility.

## Features

- Upload resume PDFs
- Resume content extraction
- ATS score analysis
- Skill gap identification
- AI-generated feedback and recommendations
- User-friendly Streamlit interface

## Tech Stack

- Python
- Streamlit
- PyPDF2
- AI/LLM Integration
- Pandas
- Git & GitHub

## Installation

1. Clone the repository

```bash
git clone https://github.com/Kashishdoaj/AI-Resume_Analyser.git
```

2. Navigate to the project folder

```bash
cd AI-Resume_Analyser
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
